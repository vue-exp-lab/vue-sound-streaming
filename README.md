# vue-ssr-starter-kit

> A Vue.js project with vue 2.0, vue-router and vuex starter kit for server side rendering. 

## Node Version Requirement

```bash
node 8.10.0
```

## Build Setup

``` bash
npm install
npm run build
npm start
```

## Development Setup

```bash
npm install
npm run dev
```
